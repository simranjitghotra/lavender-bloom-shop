**Lavender Bloom** is a Django-based eCommerce web application for flowers. The website combines a soft lavender theme with a user-friendly interface for browsing, purchasing, and customizing flower bouquets. This project is fully dynamic and designed for both customers and admin users.

## **Features**

### User Features
- Browse all flowers with images, names, prices, and descriptions.
- Filter flowers by category (Love, Friendship, Anniversary, etc.).
- View detailed product pages with:
  - Story
  - When to give
  - Pair with suggestions
  - Best for
  - Scent profile
  - Symbolism
- Add flowers to Cart and Wishlist.
- Make custom bouquets with form-based suggestions.
- User authentication (Login / Signup / Logout).

### Admin Features
- Add, Update, Delete products from admin panel.
- Upload images and product details dynamically.
- Manage orders and user interactions.
